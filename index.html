About me

Over the past 8 years, I specialized in network incident management — responding to outages, diagnosing slowdowns, and ensuring critical systems stayed online in large enterprise environments. As I explored new career paths, I noticed a recurring theme: many roles required scripting or development skills. I picked up Python and was immediately hooked — I looked forward to every new study session. It quickly became clear that software development was the path I wanted to pursue long term.

I began focusing more deliberately on software engineering, shifting my learning to the .NET ecosystem. I first built a strong foundation in data structures and algorithms to ensure I understood core programming principles. After that, I developed several backend applications using C# and .NET, gaining hands-on experience with APIs, data access layers, and service architecture. Currently, I’m expanding into frontend technologies like Blazor and Razor Pages, aiming to become a well-rounded full-stack .NET developer.

Projects

Shift Logger

A .NET project showcasing both API creation and consumption using ASP.NET Core Minimal APIs. The API handles data persistence with Entity Framework Core, including relational data modeling. Functionality includes creating and managing shift types, assigned shifts, and workers. The API was thoroughly using Swagger and Postman to ensure reliability and usability.

Excel Reader

A lightweight Excel and CSV reader built with EPPlus and CsvHelper, designed to import single-sheet dynamic data into a database using Dapper, with support for reading and exporting .xlsx and .csv files.

Sport Results Notifier

A .NET project that scrapes structured data directly from the web using the HTML Agility Pack. The extracted information is then formatted and automatically sent via email, enabling efficient data collection and reporting.

Excercise Tracker

A lightweight C# console application for tracking two types of exercises: weight training and cardio. Each exercise type uses a different ORM for data persistence — Entity Framework Core for weight training and Dapper for cardio — showcasing a flexible and modular design.

Drinks Menu

A .NET project focused on consuming an external API to display a restaurant’s drinks menu. Users can browse drinks by category and view detailed information for each item. The application features a user-friendly, terminal-based interface built with Spectre.Console for an enhanced console experience.
